## Overview
This is a C++ program that demonstrates a bookstore system. There are two modules including admin and customer module.
### Customer Module's Features
1. Display All Available Books
2. Search for Specific Books
3. Sort Books
4. Checkout the Shopping Cart
5. Display the Transaction History
6. Search Orders
7. Sort Orders
8. Change Password
9. Logout
### Admin Modules's Features
1. Add New Books
2. Delete Books
3. Update Information on Existing Books
4. Display All Products and Inventory
5. Search for Specific Books
6. Sort Book
7. Display All Customer Orders
8. Search for Specific Customer Orders
9. Sort Orders
10. Edit Order Status
11. Sales Report
12. Add New Admin
13. Logout

## Prerequisites
Before begins, ensure you have the following installed:
1. C++ compiler (VS Code, Codeblock for optimal performance)

## Running Steps
1. Run the main.cpp file inside any compiler to execute the program

## Files Included
1. users.txt to store customer and admin information (Role 1 = Customer | Role 2 = Admin)
2. books.txt to store books information
3. orders.txt to store orders total and status
4. orderdetails.txt to store placed items and quantity for each order
5. cart.txt to store the books that selected by users
